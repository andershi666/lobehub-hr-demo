## 多元化用工与特殊人群合规（新增专题）

### 专题一：多元化用工合规管理

现代企业采用多种用工模式，每种模式有不同的法律关系和合规要求：

#### 1. 劳务派遣合规

```markdown
## 劳务派遣合规要点

### 法定限制
| 限制项 | 法定要求 | 违规后果 | 合规措施 |
|--------|---------|---------|---------|
| 适用岗位 | 仅限"三性"岗位（临时性、辅助性、替代性） | 视为违法派遣,需转正 | 明确界定辅助岗位清单 |
| 派遣比例 | ≤10%（用工总量） | 劳动监察处罚 | 季度监控派遣比例 |
| 同工同酬 | 派遣员工与正式员工同等待遇 | 劳动争议风险 | 建立统一薪酬标准 |
| 派遣期限 | 不得超过合理期限 | 可能被认定为劳动关系 | 定期评估转正可能 |

### 合规检查清单
- [ ] 派遣岗位是否属于"三性"岗位？
- [ ] 当前派遣比例是否≤10%？
- [ ] 派遣员工薪酬是否与正式员工持平？
- [ ] 派遣协议是否与合法派遣公司签订？
- [ ] 是否存在"假派遣真用工"（实际管理指挥）？

### 风险场景
❌ **高风险操作**：
- 核心业务岗位使用派遣（如研发、销售）
- 派遣比例超过15%
- 派遣员工薪酬明显低于正式员工

✅ **合规操作**：
- 仅在行政、后勤等辅助岗位使用派遣
- 严格控制在10%以内
- 确保同工同酬，定期审查
```

#### 2. 非全日制用工合规

```markdown
## 非全日制用工规则

### 法定标准
**认定标准**：
- 日工作时间 ≤ 4小时
- 周工作时间 ≤ 24小时
- 小时计酬，定期结算

### 合规要点
| 合规项 | 非全日制规则 | 全日制规则 | 注意事项 |
|--------|------------|-----------|---------|
| 合同形式 | 口头或书面均可 | 必须书面 | 建议书面，明确约定 |
| 试用期 | 不得约定试用期 | 可约定试用期 | 违反无效 |
| 社保缴纳 | 仅需缴纳工伤保险 | 五险一金 | 养老医疗可不缴 |
| 经济补偿 | 解除无需经济补偿 | 需支付N | 大幅降低用工成本 |
| 竞业限制 | 不得约定 | 可约定 | 约定无效 |

### 风险识别
⚠️ **假非全日制风险**：
实际每日工作超过4小时，但按非全日制申报
→ 可能被认定为全日制，补缴社保+经济补偿

✅ **规避措施**：
- 严格记录工作时长（考勤系统）
- 合理安排任务量，确保4小时内可完成
- 按小时计薪，保留支付记录
```

#### 3. 业务外包合规

```markdown
## 业务外包 vs 劳务派遣区分

### 核心区别
**业务外包（合法且低风险）**：
```
发包方 ──业务承包协议──> 承包公司
                         │
                         └──劳动合同──> 外包人员

关键：发包方不直接管理外包人员
```

**劳务派遣（受限且高风险）**：
```
用工单位 ──派遣协议──> 派遣公司
   │                    │
   └───直接管理───────> 派遣人员 ──劳动合同

关键：用工单位直接管理派遣人员
```

### 合规判定标准
**真正的业务外包满足**：
- ✅ 发包方不直接管理外包人员（人不归我管）
- ✅ 外包公司独立完成业务（业务独立负责）
- ✅ 按成果验收付费（不是按工时）
- ✅ 外包人员使用自己的工具设备
- ✅ 外包公司承担业务风险

**判定为派遣的风险信号**：
- ❌ 发包方直接给外包人员分配任务
- ❌ 外包人员使用发包方的邮箱、工卡
- ❌ 外包人员与正式员工混编管理
- ❌ 按人头按月支付费用（而非按成果）
```

#### 4. 平台用工合规

```markdown
## 平台用工法律关系判定

### 三种法律关系
**关系一：劳动关系**（最严格，成本最高）
判定标准：
- 平台直接管理指挥（固定上班时间、KPI考核）
- 劳动者无自主性（必须接单、不能自由安排）
- 专属服务（不能同时服务多平台）
- 平台提供工具设备（如外卖箱、工作车）

**关系二：劳务关系**（中等成本）
判定标准：
- 劳动者有一定自主性
- 按单结算，非固定薪资
- 可同时服务多平台
- 但受平台一定管理约束

**关系三：合作关系**（最低成本）
判定标准：
- 劳动者完全自主
- 仅提供信息平台
- 不管理不考核
- 纯按成果分成

### 合规建议
**降低被认定为劳动关系的措施**：
- ✅ 给予劳动者充分接单自由（不强制接单）
- ✅ 不设固定工作时间
- ✅ 允许同时服务多平台
- ✅ 不提供专属工具设备
- ✅ 签订合作协议而非劳动合同
- ✅ 购买商业保险而非社保

**高风险操作**：
- ❌ 强制上下班打卡
- ❌ 设置接单率KPI并处罚
- ❌ 要求穿统一服装、使用统一设备
- ❌ 禁止同时服务其他平台
```

### 专题二：特殊人群保护合规

#### 1. 女职工"三期"保护

```markdown
## 女职工孕期、产期、哺乳期特殊保护

### 法律禁止事项
| 禁止行为 | 法律依据 | 违反后果 | 例外情况 |
|---------|---------|---------|---------|
| 解除劳动合同 | 《劳动合同法》42条 | 违法解除,赔偿2N | 严重违纪、刑事犯罪 |
| 降低工资待遇 | 《女职工劳动保护特别规定》 | 劳动争议败诉 | 协商一致 |
| 安排夜班加班 | 同上 | 行政处罚 | 无 |
| 从事禁忌劳动 | 同上 | 行政处罚+赔偿 | 无 |

### 详细保护规则

**孕期保护**（怀孕开始至生产）：
- ✓ 不得安排国家规定的孕期禁忌劳动（高空、接触有毒物质等）
- ✓ 产前检查时间计入工作时间
- ✓ 不能胜任原工作，应调岗且不降薪
- ✓ 劳动合同到期自动顺延至哺乳期结束

**产期保护**（生产及产假期间）：
- ✓ 产假98天（难产+15天，多胞胎每多一个+15天）
- ✓ 各地有延长生育假（如北京+60天）
- ✓ 产假期间工资照发（或生育津贴）
- ✓ 配偶陪产假（各地5-30天不等）

**哺乳期保护**（产后1年内）：
- ✓ 每天1小时哺乳时间（可合并使用，多胞胎增加）
- ✓ 不得安排夜班、加班、出差
- ✓ 不得安排哺乳期禁忌劳动
- ✓ 劳动合同顺延至哺乳期结束（1年）

### 常见违规与纠正

❌ **错误操作1**：孕期员工绩效不佳，直接解雇
✅ **正确操作**：调岗至适合岗位，工资不降低，顺延合同

❌ **错误操作2**：产假期间合同到期，通知不续签
✅ **正确操作**：合同自动顺延至哺乳期结束（产后1年），再决定是否续签

❌ **错误操作3**：哺乳期员工不同意调岗，按旷工处理
✅ **正确操作**：除非严重违纪，哺乳期禁止解除，协商调岗或灵活安排
```

#### 2. 残疾人用工合规

```markdown
## 残疾人按比例就业制度

### 法定要求
**按比例安置**：
- 用人单位应按不低于本单位在职职工总数1.5%的比例安排残疾人就业
- 未达标：缴纳残疾人就业保障金

### 计算示例
```
公司员工总数：300人
应安排残疾人：300 × 1.5% = 4.5人 → 向上取整 = 5人

实际安排：2人
未达标人数：5 - 2 = 3人
保障金计算：3人 × 当地上年度平均工资 × 缴费比例
```

### 合规要点
- ✅ 残疾证真实有效（定期核查）
- ✅ 真实用工（非挂靠）
- ✅ 岗位适配（避免禁忌岗位）
- ✅ 无障碍环境（必要时提供便利设施）
- ✅ 平等对待（薪酬、晋升、培训）

### 典型风险
⚠️ **虚假安置**：
- 挂靠残疾证但不实际工作
- 按最低标准缴社保逃避保障金
→ 发现后补缴保障金+罚款

✅ **合规安置**：
- 真实劳动合同
- 正常缴纳社保
- 适配岗位（如文员、客服、轻体力劳动）
```

#### 4. 医疗期员工管理

```markdown
## 医疗期管理规定

### 医疗期计算规则
| 工作年限 | 医疗期长度 | 说明 |
|---------|-----------|------|
| <5年 | 3个月 | 按6个月内累计病休时间计算 |
| 5-10年 | 6个月 | 按12个月内累计病休时间计算 |
| 10-15年 | 9个月 | 按15个月内累计病休时间计算 |
| 15-20年 | 12个月 | 按18个月内累计病休时间计算 |
| ≥20年 | 18-24个月 | 按24-30个月内累计计算 |

### 医疗期待遇
**工资支付**：
- 不低于当地最低工资的80%
- 各地具体规定不同（如北京：基本工资100%发放）

**合同处理**：
- 医疗期内合同到期：顺延至医疗期结束
- 医疗期内不得解除劳动合同

### 医疗期满处理
**医疗期满后的合规处理流程**：
```
医疗期满
    ↓
安排复工体检（确认能否从事原工作）
    ↓
┌─────────┴─────────┐
│能从事原工作│    │不能从事原工作│
└─────────┬─────────┘
          ↓              ↓
   正常复工        调整到其他岗位
                          ↓
                  仍不能胜任（30天后）
                          ↓
                  提前30天通知+支付N
```

### 常见违规
❌ **错误操作**：病假3个月，按旷工处理并解雇
✅ **正确操作**：核查是否在医疗期内，医疗期内受法律保护，不得解除

❌ **错误操作**：医疗期满立即解雇
✅ **正确操作**：需安排复工体检→安排合适岗位→仍不能胜任→提前30天通知+N
```

## 专业建议

### 劳动合规的黄金法则
1. **程序正义**：结果可能无法控制,但程序必须合法
2. **证据意识**：没有证据=没有发生
3. **提前咨询**：事前咨询成本远低于事后补救
4. **一致性原则**：同样情况同样处理,避免差别对待
5. **书面化习惯**：口头约定有风险,重要事项必须书面

### 高风险场景特别提示

**调岗降薪**：
- ❌ 单方面通知调岗降薪（极高风险）
- ✅ 协商一致+书面确认（低风险）

**绩效解雇**：
- ❌ 绩效不达标直接解雇（必败）
- ✅ 绩效改进计划+培训+充分机会+完整证据链（可控）

**规章制度**：
- ❌ 公司单方面制定（可能无效）
- ✅ 民主程序+公示告知+员工签收（有效）
